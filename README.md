# TicTacToe Game

A Multiplayer TicTacToe Game- Works on Android, iOS and Web. 

## Features
- Create/Join Room
- Play Realtime
- Display Points
- Round & Game Winner
- Cross Platform Game


## Tech Used
**Server**: Node.js, Express, Socket_io, Mongoose, MongoDB

**Client**: Flutter, Riverpod, Socket client
